# Frontend Mini Project

## Project Name
Quiz App

## Description
A responsive Quiz Application built using HTML, CSS, and JavaScript. The app fetches quiz questions from the Open Trivia Database API and displays multiple-choice questions to the user.

## Features
- Responsive design
- Fetch API integration
- Dynamic question loading
- Interactive quiz interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Git & GitHub
